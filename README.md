# Precsys Front

Interface web do **sistema de carteira de precatórios**. Este repositório contém o **front-end** responsável pelas páginas de consulta, visualização, inserção, atualização e remoção de dados.

**Stack principal:** React + Tailwind CSS.

## Tecnologias

- **React**
- **Tailwind CSS**
- **Webpack**
- **Axios/Fetch**
